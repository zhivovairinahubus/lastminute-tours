# LastMinute Tours 🏖️

> Сервис для спонтанных путешественников — туры на ближайшие выходные по лучшей цене

## О проекте

LastMinute Tours — это MVP-сервис для поиска горящих туров с вылетом в ближайшие 7 дней. Пользователь вводит только город вылета и бюджет, а сервис автоматически находит топ-3 самых выгодных предложения и генерирует для каждого AI-описание в стиле «что успеете увидеть и стоит ли брать».

## Целевая аудитория

Люди, которые хотят спонтанно уехать на выходные, но не знают куда. Возраст 25–45 лет, работают, ценят время и простоту выбора.

## Технологический стек

### Frontend
- **React 18** + **Vite** — быстрый SPA
- **TypeScript** — типизация
- **TailwindCSS** — стилизация
- **TanStack React Query** — управление состоянием и кэширование запросов

### Backend
- **Node.js** + **Express 5** — API-сервер
- **TypeScript** — типизация
- **OpenAPI 3.1** — контракт API (code-first через Orval)

### AI
- **OpenAI GPT (gpt-5-mini)** через Replit AI Integrations — генерация описаний туров
- Промпт: получить краткое описание что увидеть за N ночей + вердикт "стоит брать"

### Данные о турах
- Интеграция с **Level.Travel API** (демо-режим с реалистичными данными)
- Туры сортируются по цене, показываются топ-3


## API Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/healthz` | Проверка здоровья сервиса |
| GET | `/api/tours/cities` | Список городов вылета |
| POST | `/api/tours/search` | Поиск туров |

### POST /api/tours/search

**Тело запроса:**
```json
{
  "departureCity": "Москва",
  "budget": 80000,
  "adults": 2
}
```

**Ответ:**
```json
{
  "tours": [...],
  "departureCity": "Москва",
  "budget": 80000,
  "totalFound": 10
}
```
